***************

.. only:: html

   document to the ARKODE User Documentation. The example programs demonstrate a
   large variety (but not exhaustive set) of ARKODE options and can be used as
   templates for new problems. Further information on the ARKODE package itself
   may be found in the ARKODE user guide.
