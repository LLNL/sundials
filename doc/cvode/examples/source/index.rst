.. _CVODE.Examples:

**************
CVODE Examples
**************

.. only:: html

   The CVODE examples documentation is intended to serve as a companion document
   to the CVODE User Documentation. These example programs demonstrate a large
   variety (but not exhaustive set) of CVODE options and can be used as
   templates for new problems. Further information on the CVODE package itself
   may be found in the CVODE user guide.
